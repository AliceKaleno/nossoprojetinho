# 📱 Nosso Projetinho

> Projeto acadêmico desenvolvido em dupla durante as aulas de Programação Mobile utilizando React Native e Expo.

---

## 📖 Sobre o Projeto

O "Nosso Projetinho" foi desenvolvido como atividade prática da disciplina de Programação, com o objetivo de explorar conceitos de desenvolvimento mobile utilizando React Native.

Durante a construção do projeto foram estudados recursos como navegação entre telas, organização de componentes, roteamento moderno com Expo Router e personalização da interface.

Mais do que o resultado final, este projeto representou uma importante etapa de aprendizado no desenvolvimento de aplicações móveis.

---
